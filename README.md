## Installation
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/lehongngan98/Enggo_App.git
   ```

2. Install dependencies:
   
   ```bash
   npm install
   ```

3. Start the project:
  
   ```bash
   npm start
   ```
